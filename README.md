# my_typora_theme
在typora Vue 主题上做了部分修改

1. 标题自动标号
2. 代码块 仿MAC窗口样式
3. 修改多级标题大小使其与正文明显区分
4. 从[少数派的主题](https://github.com/sheilaCat/typora-theme-css)里偷了图片标签处理：
   - zoom：自动缩放
   - shadow：给图片增加类似mac窗口的阴影
   - zoom-shadow / shadow-zoom
